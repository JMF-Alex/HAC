## About the Project

**HAC** is A simple autoclicker made with Python and tkinter

### Features

- Customizable interval (in seconds)
- Hotkey support

## Built With

This project was developed using:

- **Python** for backend logic
- **Tkinter** for Python GUI applications

Repository: [github.com/JMF-Alex/HAC](https://github.com/JMF-Alex/HAC)

## Getting Started

### Prerequisites

Install dependencies:
```bash
pip install pillow keyboard
```

### Installation

1. Clone the repository:

```bash
git clone https://github.com/JMF-Alex/HAC.git
```

2. Enter the project directory:

```bash
cd HAC
```

3. Open `index.html` in your browser.

That's it â€” the app runs entirely locally!

## Usage

Once opened, you can:
